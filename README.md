# technova-site

This is a web development project created using HTML5 and CSS3. It serves as a showcase of fundamental web layout techniques and modern design principles.

## Project Overview
TechNova Solutions is a concept website designed to demonstrate the structure of a professional digital agency landing page. The project highlights:
- Responsive Header: A clean, flex-based navigation bar.
- Hero Section: An engaging introductory section featuring a modern CSS gradient.
- Service Cards: Interactive cards that use CSS transitions (`transform: translateY`) for a smooth hover effect.
- Footer: A simple, structured footer with copyright information.

## Technologies Used
- HTML5: For document structure and semantic elements.
- CSS3: For styling, layout (Flexbox), box shadows, and hover animations.
